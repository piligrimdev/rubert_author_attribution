# О сервисе

Сервис **определения авторства** помогает определить, кто мог написать текст фрагмента, сравнивая его по стилю с загруженными текстами чьи авторы известны.

## Возможности

- **Определения авторства текста**
- **База авторов** — добавление писателей, текстов и жанров; импорт корпуса из `.csv` талбицы.
- **Расчет стилистических метрик** — графики и сравнение авторов по лексике, синтаксису и читабельности.

## Как это работает

1. Пользователь добавляет в базу авторов и тексты для них.

2. Для загруженных пользователем текстов нейросетевая модель строит векторные представления, которые отражают стиль текста.

3. Для неизвестного фрагмента также вычисляется "вектор стиля" выполняется поиск **ближайших по стилю текстов** в базе.

4. Результат — список похожих текстов предполагаемый автор с оценкой схожести.

> Совет: для надёжности определения используйте фрагменты **не короче 50 символов** и при необходимости ограничьте набор авторов для сравнения.
